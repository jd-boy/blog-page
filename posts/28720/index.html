javac

先从 javac 的编译选项 -source、 -target 说起：

  • -source：指定使用什么版本的 JDK 语法编译源代码。
  • -target：指定生成特定于某个 JDK 版本的 class 文件。高版本的 class 文件不被低版本支持，因此需要该项。注意，最好设置 -bootclasspath 指定对应 JDK 版本的 boot classes 文件，否则即使设置了-target 也不能在指定版本上运行 class 文件
    一般情况下，-target 与 -source 设置一致，可以不用设置 -target，但最好设置它。

maven
